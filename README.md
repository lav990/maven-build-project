# Maven Build Automation Project

## Objective
To automate the build of a Java application using Apache Maven and generate a build artifact (JAR file).

## Tools Used
- Git  
- GitHub  
- Java (JDK)  
- Apache Maven  

## Build Steps
1. Clone the repository  
2. Navigate to the project directory  
3. Run the command:  
   mvn clean package  
4. Verify the generated JAR file inside the target folder  

## Output
JAR file generated successfully in the target directory.

## DevOps Role
Application source code was taken from an open-source sample project for learning purposes. 
My role focused on:
- Creating a Maven project  
- Managing dependencies using pom.xml  
- Automating the build process using Maven  
- Generating the build artifact (JAR file)
